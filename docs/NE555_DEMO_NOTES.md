# NE555 LED Blinker Demo

File: `examples/ne555_led_blinker_demo.pcbforge`

This project is a small PCB demo used to test PCB Forge features:
- NE555 timer IC with named pins
- LED output path
- timing resistor/capacitor
- power connector
- GND copper zone
- silkscreen labels
- mounting holes
- BOM metadata
- traces, netlist and DRC

It is not meant to be a verified production electronics design. It is a functional QA board layout for testing the editor.
